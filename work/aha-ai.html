Overview

An AI-driven learning platform helping students prepare more effectively for English proficiency exams.

My focus was enhancing usability, visual consistency and system scalability across web and app. Key responsibilities included establishing a unified design system, standardizing UI patterns, optimizing cross-platform UX flows, and supporting the product team through prototypes and design QA.

The System

The previous system had drifted out of sync across platforms. I rebuilt it on a consistent spacing and typography foundation so that every screen — from practice to exam mode — felt like one product.

間距系統重點：字級 16px 行高 150%（24px）；標題與段落間距 8px、段落之間 24px；外框與主內容間距 32px；元件內距統一 8px。
