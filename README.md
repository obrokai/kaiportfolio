# kaiportfolio

Personal portfolio of **Mike (kaidesignworks)** — UI/UX Designer crafting clear, intuitive digital experiences.

Single-page, fully static site. No build step required.

## Local preview
Open `index.html` in a browser, or run a tiny server:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Deploy (Vercel)
This is a static site, so Vercel needs no configuration:

1. Push this repo to GitHub.
2. In Vercel, **Add New → Project** and import `obrokai/kaiportfolio`.
3. Framework Preset: **Other**. Build Command: *(empty)*. Output Directory: *(leave default / `.`)*.
4. Deploy.

## Tech
- Hand-written HTML + CSS + vanilla JS
- Google Fonts: Inter, Inter Tight, Instrument Serif
- Light, editorial design system
